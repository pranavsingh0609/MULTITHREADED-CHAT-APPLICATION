##Task Description
The task is to develop a multithreaded chat application in Java using a client-server architecture. The server should use sockets to listen for incoming connections and handle multiple clients simultaneously by creating separate threads for each connected client. Each client should be able to send messages to the server, which will then broadcast those messages to all other connected clients in real time. The final deliverable is a fully functional chat system consisting of one server program and multiple client programs that can communicate with each other concurrently.
